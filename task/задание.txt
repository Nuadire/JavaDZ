1. Разработать программу на Java:

1. Графический интерфейс с применением Swing.
2. Режим редактирования таблицы БД, содержащей сведения о персонале (наименование таблицы, список колонок и их тип придумать самим)
3. Режим экспорта содержимого таблицы в формат XML и импорта из формата XML в таблицу (формат XML придумать самим)
4. Протоколирование работы системы (с применением java.util.logging). Должны протоколироваться моменты запуска и завершения работы приложения, а также импорта и экспорта данных (с указанием количества загруженных/выгруженных данных)
5. Кадр настроек системы (имя драйвера БД, строка подключения, имя пользователя, пароль, местоположение файла настроек java.util.logging)

2. Собрать программу в виде выполнимого файла.

3. Включить в программу возможность запуска импорта и экспорта данных через командную строку:
java -jar <ваш_архив> -import <имя_файла_XML>
java -jar <ваш_архив> -export <имя_файла_XML>

При использовании командной строки графическую оболочку запускать не нужно.
